# Python Калькулятор & Конвертер

Мой первый Python проект.

## Особенности
- Обычная арифметика: Сложить, Отнять, Умножить, Разделить, Возвести в степень, Корень.
- Тригонометрия: Синус, Косинус, Тангенс, Котангенс (ответ в радианах).
- Дополнительный инструмент: Конвертер радиан в градусы.
- Чистая модульная архитектура.
- Комментарии на русском внутри кода

## Как запустить
1. Убедитесь, что установлен Python.
2. Запустите `Simple_calculator.py` чтобы воспользоваться калькулятором.
3. Запустите `Radians_to_Degree_converter.py` для конвертации радиан.
